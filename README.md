# Snake Game With Pygame
 This Snake Game Is Made Using Python 3.6.5 & Pygame Module. Hope You Will Like The Game.. and also u can use the code to make your own version of game.  To play the game, simply double click on the Snake.exe. Installing The Pygame Module:  Open Terminal/CMD Type pip install pygame/ pip3 install pygame OR  Open Terminal or CMD, then Type pip install -r <path to the game>  Dont Delete Any Files..
